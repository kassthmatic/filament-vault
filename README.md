# Filament Vault

**Tech Stack:** React, Node.js, Express, MongoDB, Jest

## Overview
Filament Vault is a full-stack web application designed to manage and explore 3D printing filament products. The application allows users to browse, filter, favorite, and manage filament data through a clean and responsive interface.

## Key Features
- RESTful API built with Node.js & Express
- MongoDB database integration
- React frontend with component-based architecture
- Save-to-favorites functionality
- JWT-based authentication
- Jest unit testing
